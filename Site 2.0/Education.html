Unity

Unity - это популярный движок для создания игр и других интерактивных приложений. С Unity вы можете разрабатывать игры для разных платформ, таких как Windows, Mac, Linux, Android, iOS, Web и других. Unity поддерживает несколько языков программирования, в том числе C# и JavaScript.

Для начала работы с Unity вам понадобятся следующие вещи:

  • Скачать и установить Unity Hub с официального сайта: https://unity3d.com/get-unity/download
  • Создать или войти в свою учетную запись Unity на сайте: https://id.unity.com
  • Активировать лицензию Unity в Unity Hub: https://docs.unity3d.com/Manual/OnlineActivationGuide.html
  • Скачать и установить нужную версию Unity Editor в Unity Hub: https://docs.unity3d.com/Manual/GettingStartedInstallingHub.html
  • Создать новый проект или открыть существующий в Unity Hub: https://docs.unity3d.com/Manual/GettingStartedCreatingProjects.html

Для изучения основ Unity вы можете посмотреть следующие учебные материалы:

  • Официальные руководства и документация: https://docs.unity3d.com/Manual/index.html
  • Официальные видеоуроки: https://learn.unity.com
  • Официальный блог: https://blog.unity.com
  • Официальный форум: https://forum.unity.com
  • Официальный канал на YouTube: https://www.youtube.com/user/Unity3D

Для проверки своих знаний по Unity вы можете пройти следующие тесты:

  • Тест по основам Unity: https://connect.unity.com/assessment/5c4f8c0cedbc2a0020e4d79f
  • Тест по скриптингу в Unity: https://connect.unity.com/assessment/5c4f8c0cedbc2a0020e4d7a0
  • Тест по графике в Unity: https://connect.unity.com/assessment/5c4f8c0cedbc2a0020e4d7a1
  • Тест по анимации в Unity: https://connect.unity.com/assessment/5c4f8c0cedbc2a0020e4d7a2
  • Тест по физике в Unity: https://connect.unity.com/assessment/5c4f8c0cedbc2a0020e4d7a3

GitHub

GitHub - это платформа для хранения, управления и совместной работы над кодом. С GitHub вы можете создавать и хранить репозитории для ваших проектов, синхронизировать их с локальными копиями, делиться и получать обратную связь от других разработчиков, вносить изменения и объединять их с помощью коммитов и пул-реквестов, а также использовать различные инструменты и сервисы, интегрированные с GitHub.

Для начала работы с GitHub вам понадобятся следующие вещи:

  • Создать или войти в свою учетную запись GitHub на сайте: https://github.com
  • Создать новый репозиторий или склонировать существующий на свой компьютер: https://docs.github.com/en/get-started/quickstart/create-a-repo
  • Установить и настроить Git на своем компьютере: https://git-scm.com/downloads
  • Связать свой локальный репозиторий с удаленным на GitHub: https://docs.github.com/en/get-started/quickstart/set-up-git
